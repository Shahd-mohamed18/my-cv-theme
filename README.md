## Custom Modules

### Hero Section
- Displays the main introduction
- Includes title, subtitle, background image, and CTA button

### Skills Section
- Shows a list of skills
- Uses progress bars to display skill levels

### Projects Gallery
- Displays projects in a grid layout
- Each project includes image, description, and link

### Contact Section
- Shows contact information and social links
- Option to show or hide a contact form

## Deployment
1. Upload files to HubSpot Design Manager  
2. Publish modules and template  
3. Create a page using the portfolio template  
4. Edit content from the page editor  
5. Preview and publish the page
